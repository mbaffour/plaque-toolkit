# Unified 'plaqueapp' env — the SINGLE environment that runs EVERYTHING (PST + PlaqSeg +
# learned classifier) in ONE process, and the one packaged into the self-contained
# installer (build/plaque_app_full.spec).
#
# The key insight: numpy 1.26 satisfies BOTH PST's `numpy<2` pin (it used np.warnings,
# removed in numpy 2.0) AND PyTorch/ultralytics — so Precise no longer needs two conda
# envs. numpy MUST be pinned LAST (opencv-python/ultralytics pull numpy>=2 metadata, but
# cv2 4.13 runs fine on numpy 1.26 at runtime — only a harmless pip metadata warning).
#
# Recipe (Windows, verified):
#   conda create -n plaqueapp python=3.10 -y
#   conda run -n plaqueapp python -m pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
#   conda run -n plaqueapp python -m pip install PySide6 opencv-python pandas matplotlib pillow-heif imutils scikit-image ultralytics pyinstaller
#   conda run -n plaqueapp python -m pip install "numpy==1.26.4"   # <-- pin LAST
#
