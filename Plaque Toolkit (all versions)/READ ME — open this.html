🧫 Plaque Toolkit — all versions

Every way to measure your plaques, in one place. Pick a version below, or build an exact command. All launchers here run the real tools in the parent folder, so you can keep this folder as your home base.

Validated

Original published tool

Reproduces Trofimova & Jaschke 2021 exactly. This is the one you cite as a published, peer-reviewed method. Use it for the numbers that go in your paper.

Enhanced

Current tool + GUI + app

Same engine, plus turbidity, watershed, the click-editor, the dish overlay, and the sensitive (tiny-plaque) mode. Great for exploring — but the extras are not the published validation.

For publication, in one line: use the Original published tool as your citable engine; anything you change (especially sensitive mode) you must validate yourself on your own plates and report separately — it does not inherit the published validation. The sensitive mode trades precision for recall, so verify its detections by eye.

Which version should I use?

The versions

1 · Measure plaques Enhanced

1 - Measure plaques (current).bat

Drag one or more photos onto it → annotated image + CSV (size and turbidity columns).

When: quick measurement of a few photos with the current engine.

2 · Edit plaques — click GUI Enhanced

2 - Edit plaques - click GUI.bat

Opens a window: zoom to find missed plaques, left-drag to add, t+click to trace, right-click to remove, Sensitive ON/OFF button, orange dish circle for a calibration check.

When: the auto-detector missed some and you want to correct by hand.

3 · Batch a folder Enhanced

3 - Batch a folder (current).bat

Drag a folder of plate photos → one data-green-<plate>.csv each + a combined summary.csv. Name each file after its plate.

When: many plates at once with the current engine.

4 · ORIGINAL tool — one plate Validated

4 - ORIGINAL published tool - one plate.bat

Exact published algorithm on a single photo. Diameters within ≤0.04 mm of the literal original.

When: the value goes in your paper. This is the citable one.

5 · ORIGINAL tool — batch Validated

5 - ORIGINAL published tool - batch.bat

Exact published algorithm over a whole folder → CSV per plate + summary.

When: publishable numbers for many plates.

6 · Compare turbidity Enhanced

6 - Compare turbidity (cross-phage).bat

A folder of phages (best: transmitted-light photos + a blank-agar reference) → optical-density turbidity, clarity, optional titer, per-phage stats & figures.

When: comparing plaque clarity across phages imaged the same day.

7 · Plaque Toolkit app Enhanced

7 - Plaque Toolkit app (everything).bat

One window: Measure tab (open → detect → edit → save, with a sensitive checkbox + Re-detect) and a Compare turbidity tab. No command line.

When: you want everything in a single point-and-click app.

First time? / setup

  1. These launchers need the plaque conda environment (Python 3.9). It's already set up on this machine.
  2. To measure your own plates: put photos in the plates folder (or just drag them onto a launcher). iPhone HEIC is supported.
  3. Results land in the out folder next to the tool (annotated image + CSV).
