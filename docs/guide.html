Quick start

Measure bacteriophage plaques — size, turbidity, count, titer — from Petri‑dish photos. iPhone HEIC works directly. Built on the validated Plaque Size Tool.

📸 One image → the numbers

Drag your image onto Measure Plaques.bat, press N, type 100 (dish mm). Read out\data-green-<image>.csv.

🧪 Compare phages (publishable)

Name images per phage, drag the folder onto Compare Turbidity.bat. Read out_turbidity\per_phage.csv + the figures.

For real cross‑phage turbidity you want the turbidity tool with a blank‑agar reference — see Turbidity explained.

plaque_size_tool.py — automatic size + count + turbidity index

Fast measurement of one image or a whole folder. Outputs size (px & mm), count, brightness, and a within‑plate turbidity index.

Launcher: drag image(s) onto Measure Plaques.bat → choose Normal/Small → dish mm.

conda run -n plaque python "plaque_size_tool.py" -i "IMAGE" -p 100

plaque_turbidity.py — cross‑phage turbidity (the publishable one)

Batch optical‑density turbidity anchored to shared references, plus clarity class, count + PFU/mL titer, per‑phage stats, and figures.

Launcher: name images per phage, drag the folder onto Compare Turbidity.bat.

conda run -n plaque python "plaque_turbidity.py" -d "FOLDER" -p 100 --blank "blank.tif" --flat "flat.tif" --group-by-prefix

heic_to_tiff.py — convert HEIC to TIFF

Rarely needed (all tools read HEIC directly). Use it only if you want TIFF copies.

conda run -n plaque python "heic_to_tiff.py" -d "HEIC_FOLDER"

Turbidity explained read this

There are two turbidity numbers — use the right one:

Within‑plate index

From the size tool / GUI. 0 = clearest plaque on that plate, 1 = the lawn. Good for ranking on one plate. not cross‑plate

Optical density (OD)

From the turbidity batch tool. OD = −log₁₀(I/I₀) anchored to clear agar & lawn. comparable across phages — the publishable one.

For phage comparisons, run the turbidity tool with --blank. Without it, only a relative TRANSMITTANCE (qualitative) is reported.

Imaging protocol (for good turbidity)

Transmitted light (light box). Fixed manual exposure / white balance / focus; HDR off; prefer iPhone ProRAW over HEIC. Same dish position; lids off. In one session capture:

  1. Flat‑field (bare light box) → --flat
  2. Blank‑agar plate (no bacteria) → --blank (the key reference)
  3. optional dark frame (lens covered) → --dark
  4. all phage plates — filename = phage (T4_1,T4_2 for replicates)
  5. re‑image one plate twice → reproducibility check
Check qc.csv each run: POLARITY_OK = True, FRAC_SATURATED ≈ 0, low ILLUM_CV.

Watershed & published modes

--watershed — split touching plaques

Opt‑in. Splits fused plaques the detector drops, recovers real‑sized ones (cyan, SOURCE=watershed), no over‑segmentation. Off by default; never changes the validated result unless asked. (Classical equivalent of the deep‑learning PlaqSeg app.)

--published — exact validated output

Reproduces the published Plaque Size Tool exactly (the default applies two tiny precision corrections — 0 count change, ~0.01 mm — and the corrected value is more precise). Use --published to cite the validated tool with no caveat.

Using it in a paper

  • Report OD as the primary turbidity metric; TURBIDITY (0–1) secondary; CLARITY categorical.
  • Stats: the plate is the replicate, not the plaque — use the *_PLATE_MEAN/_SD columns, ≥3 plates/phage, non‑parametric tests or a mixed model.
  • Cite the validated tool (Trofimova & Jaschke 2021); describe turbidity as transmitted‑light relative OD vs a cell‑free agar reference. Keep run_metadata.json for provenance.
