Prove your plaque sizes agree with Fiji

Measure a sample of plaques both ways, and turn the difference into one publishable number — the bias and 95% limits of agreement. Paste your two columns below; this page does the statistics and writes the sentence.

You need ~30–50 plaques · 2–3 plates Tools Plaque Toolkit + Fiji/ImageJ Output Bland–Altman bias + 95% LoA + ICC

§1The only rule that matters

You're comparing the same physical plaque, measured two ways. That's the whole game. Excel is completely fine for this — as long as row 7 in your tool column and row 7 in your Fiji column are the same plaque.

The catch: the tool numbers plaques top‑to‑bottom, but Fiji numbers them in the order you clicked. So "#5" in each is usually a different plaque — line those up by row and your comparison is meaningless.

◑ Fiji — numbered by click order

So why does the app have a "Compare vs Fiji" button? Only to do that pairing for you — it matches each Fiji plaque to the tool's by location, so you can't line up the wrong ones. If you measure a small set and track which tool‑number you traced (write it down, or use the app's numbered map), plain Excel — or this page — gives the identical answer. Use whichever you like.

§2Do the measurements

Pick ~30–50 plaques spanning small→large across 2–3 plates. Random is good. For each plaque, record its diameter from both tools, keeping them paired.

1

Detect the platePlaque Toolkit

  • Open the plate, detect, set the dish size so the scale is calibrated (e.g. 0.039 mm/px ✓).
  • Export ▾ → Fiji registration bundle — this saves a calibrated crop to open in Fiji, plus a numbered map so you can see which plaque is which.
  • Note each chosen plaque's number and its DIAMETER_MM (or Export all the CSV and read the column).
2

Measure the same plaquesFiji / ImageJ

  • Open <name>_plate.tif (already in mm).
  • Analyze → Set Measurements → tick Area and Centroid.
  • Trace each chosen plaque yourself — Wand (click inside) or an Oval — and press t to add it to the ROI Manager. Trace your own outlines; don't load the app's RoiSet (that would just re‑measure the tool's shapes, not an independent check).
  • Measure → File → Save As → Results.csv. Fiji gives Area; this page converts it to diameter for you.
3

Get your numberthis pageor the app

  • Simplest: paste the two diameter columns into the calculator below.
  • Or in the app: Compare vs Fiji… → pick your Results.csv — it matches by position and prints the same stats.

§4What you're allowed to claim

Size is the clean claim — count is separate This validates how big the plaques are (diameter agreement). It does not validate how many — that needs whole‑plate hand counts on more plates. Report the two separately, and don't call the in‑house engines "validated methods" (only the Published engine is peer‑reviewed).

A defensible size‑agreement sentence names the sample, the bias, and the limits of agreement — which is exactly what the calculator writes for you. That mirrors how the original Plaque Size Tool was checked against manual ImageJ.
