Measuring plaques in Fiji (ImageJ)

A step‑by‑step, hands‑on guide to measuring bacteriophage plaque size in Fiji — as the independent reference to validate Plaque Toolkit (and to make scale‑bar figures).

Why do this — and what you'll measure

Fiji is your independent yardstick. You measure a handful of plaques by hand in Fiji, then compare those diameters to what Plaque Toolkit reports. That comparison (bias + limits of agreement) is the size‑validation a reviewer will ask for. Fiji also lets you crop a single plaque and add a correct scale bar for a figure.

Match definitions. Plaque Toolkit reports the area‑equivalent diameter d = 2·√(A/π). So the cleanest thing to measure in Fiji is the plaque's Area, then compute the same equivalent diameter — apples to apples. (You can also record Feret's diameter and the fit‑ellipse axes; more below.)

Keep orientation consistent. Use the same rotation/flip in Fiji and in Plaque Toolkit (the tool now has an Orient ▾ menu) so the same plaque is “#1” in both and coordinates line up.

1 · Open the image

1
File → Open… and choose your plate image. Best: export the plate from Plaque Toolkit via Export ▾ → Cropped plate for Fiji — that TIFF has the mm scale baked in, so Fiji opens it already calibrated and you can skip step 2. Check it: Image → Properties… (Ctrl+Shift+P) should show Pixel width in mm.
If you open a raw phone photo instead, its scale is unknown — do step 2. For colour plates, Image → Type → 8‑bit can make thresholding easier later (optional).

2 · Set the scale — pixels → millimetres

Skip this if you opened the tool's calibrated crop. Otherwise, tell Fiji how many pixels equal a known real distance.

1
Pick the straight‑line tool (5th icon on the toolbar).
2
Draw a line along a known distance — ideally a ruler laid in the agar plane, or across the agar diameter (not the lid!).
3
Open Analyze → Set Scale… — Fiji fills in Distance in pixels from your line. Type the Known distance and set Unit of length = mm. Tick Global to apply it to every image you open this session. Click OK.
Agar, not lid. The vendor's “90 mm / 100 mm” is the lid. Calibrate to the agar (where plaques grow, often ~85 mm) or a ruler — the same rule as in Plaque Toolkit.

3 · Choose what to measure

1
Open Analyze → Set Measurements… and tick these:
Tick thisGives you
AreaPlaque area (in mm² once calibrated) → convert to d = 2√(A/π). Your primary number.
PerimeterOutline length; also used for shape.
Fit ellipseMajor / Minor axis lengths (mm) — a direct diameter for roundish plaques.
Feret's diameterFeret (longest caliper span) + MinFeret — the max/min width.
Shape descriptorsCircularity, AR (aspect ratio) — sanity‑check round vs irregular.
Display labelPuts the image name in each row (handy across many plates).
Set Decimal places to 3 while you're here. Click OK.

4 · Measure one plaque

Pick whichever selection method you like — then press Ctrl+M (Measure) to send a row to the Results table.

Method A — Wand (auto‑trace the outline)  ·  recommended for Area
  1. Select the Wand (tracing) tool (looks like a magic wand on the toolbar).
  2. Double‑click it to set a Tolerance (start ~20–40) and mode Legacy/8‑connected.
  3. Click inside a plaque — Fiji traces its boundary. If it grabs too much/little, tweak the tolerance and click again.
  4. Ctrl+M → you get Area (+ Feret, ellipse). This is the cleanest match to the tool's area‑equivalent diameter.
Method B — Oval selection (fast, for round plaques)
  1. Pick the Oval tool. Drag an ellipse to cover the plaque; hold Shift for a circle.
  2. Nudge/resize with the handles until it hugs the clear zone.
  3. Ctrl+M → Area + (Fit ellipse) Major/Minor diameters.
Method C — Line across the diameter (quickest single number)
  1. Straight‑line tool. Draw across the widest part of the plaque, edge to edge.
  2. Ctrl+M → Length = that diameter (mm).
Fast, but it's one chord — for irregular plaques the Area method is fairer.
To convert an Area to the tool's diameter: d = 2·√(Area/π). In a spreadsheet: =2*SQRT(Area/PI()).

5 · Measure many plaques (ROI Manager)

The efficient way to do a whole plate and keep every outline.

1
Open Analyze → Tools → ROI Manager… (or just press t after a selection to create it).
2
Select a plaque (wand/oval), press t to Add [t] it to the manager. Repeat for each plaque.
3
In the ROI Manager click Measure — every ROI is measured into the Results table at once. More ▸ Save… stores the ROI set (RoiSet.zip) so you can reload and re‑measure later.
Tick Show All + Labels in the ROI Manager to see numbered outlines on the image — makes it easy to match against Plaque Toolkit's numbering.

Optional · Automatic (Analyze Particles)

Threshold + Analyze Particles — automated area for all plaques
  1. Image → Adjust → Threshold… (Ctrl+Shift+T). Drag the sliders so only the clear plaques are highlighted, then Apply.
  2. Analyze → Analyze Particles…. Set a Size (mm²) range to drop specks, a Circularity range (e.g. 0.5–1.0), tick Display results, Add to Manager, Exclude on edges.
  3. OK → every plaque is measured automatically.
On top‑lit phone photos, thresholding plaques cleanly is hard (uneven lighting, halos). The manual wand/ROI route is usually more reliable — this is exactly the problem Plaque Toolkit's detector solves.

6 · Save the results

1
In the Results window: File → Save As… → save as .csv (e.g. fiji_<plate>.csv).
2
One CSV per plate keeps things tidy for the comparison in the next step. (Results → Clear Results before the next plate.)

7 · Compare to Plaque Toolkit

Now line the two up. In Plaque Toolkit, measure the same plate and Export all → data_<plate>.csv (its DIAMETER_MM is already area‑equivalent). In your spreadsheet:

  1. Convert each Fiji Area to a diameter: =2*SQRT(Area/PI()).
  2. Match plaques by number/position; put Fiji d next to the tool's DIAMETER_MM.
  3. Compute the difference (tool − Fiji) per plaque → mean bias + 95% limits (Bland–Altman). That's your size‑agreement result.
This paragraph — “tool vs Fiji, bias X mm, limits Y–Z” — is exactly the size validation for your Methods. See docs/METHODS_TEMPLATE.md and docs/PUBLISHING_CHECKLIST.md.

FAQ

My measurements are in pixels, not mm.

The scale isn't set. Open the tool's calibrated crop, or do step 2 (Set Scale, Unit = mm, tick Global).

Which “diameter” should I report?

To match Plaque Toolkit, use the area‑equivalent diameter from Area. Feret and ellipse‑Major are fine alternatives if you state which you used and keep it consistent.

The wand grabs the whole plate or nothing.

Double‑click the wand and adjust Tolerance; try 8‑connected/Legacy mode. Or use the Oval method.

Halo vs sharp zone?

Decide once and be consistent: trace the sharp clear zone, or include the halo. Plaque Toolkit's number is halo‑inclusive (convex hull), so trace the halo if you want the closest match — and say which in your Methods.
