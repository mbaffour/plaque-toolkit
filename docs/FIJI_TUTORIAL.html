Measuring plaques in Fiji (ImageJ)

The detailed version — plaque size and gray value / clarity (turbidity), densitometry, ROI workflows, macros, and how it all lines up with Plaque Toolkit.

Why, and the two things you measure

Fiji is your independent yardstick for validating Plaque Toolkit, and a densitometer for clarity. Two families of measurement:

FamilyFiji givesPlaque Toolkit column
Spatial (needs a mm scale)Area, Perimeter, Feret, ellipse axes → diameterAREA_MM2, DIAMETER_MM
Intensity / gray value (no scale needed)Mean/Min/Max/StdDev gray, Integrated densityMEAN_GRAY, TURBIDITY_REL
Match definitions. The tool's diameter is the area‑equivalent d = 2√(A/π), and its turbidity is plaque mean gray ÷ lawn mean gray. Measure Area and Mean gray value in Fiji and you can reproduce both exactly.
Same orientation. Rotate/flip the plate the same way in Fiji and in Plaque Toolkit (use the tool's Orient ▾ menu) so plaque #1 is the same in both.

1 · Prepare the image

1
Open: File → Open…. Best is the Plaque Toolkit export Export ▾ → Cropped plate for Fiji — a TIFF with the mm scale baked in (skip step 2). Verify with Image → Properties… (Ctrl+Shift+P).
2
Make it single‑channel for gray‑value work. Gray value is only well‑defined on one channel: Image → Type → 8‑bit (RGB→gray), or Image → Color → Split Channels and measure the channel with the best plaque contrast (often the one matching your lawn's colour). Keep a copy of the original for size/figures.
3
Flatten uneven lighting (optional, big help on phone photos): Process → Subtract Background… — rolling‑ball radius a bit larger than the biggest plaque; tick Light background if plaques are darker than the lawn. Do this on a duplicate so your raw pixels stay intact for the record.
Transmitted vs reflected light. On a light box (transmitted), a clear plaque is bright; on top light (reflected), it's usually darker than the lawn. Note which you have — it sets the sign of "clarity". You can Edit → Invert so "clearer = higher value" if that's easier to reason about.

2 · Set the scale — pixels → millimetres

Skip if you opened the calibrated crop. Otherwise:

1
Straight‑line tool → draw along a known distance (a ruler in the agar plane, or the agar diameter — not the lid).
2
Analyze → Set Scale… → Known distance + Unit = mm, tick Global, OK.

4 · Measure one plaque (size)

Select a plaque, then Ctrl+M (Measure) → a row in Results.

Method A — Wand (auto‑trace the outline) · recommended for Area
  1. Select the Wand (tracing) tool. Double‑click it to set Tolerance (start 20–40) and Legacy / 8‑connected mode.
  2. Click inside a plaque → it traces the boundary. Tweak tolerance & re‑click if it grabs too much/little.
  3. Ctrl+M → Area (+ Feret, ellipse, and — because you ticked them — Mean gray etc. of that same region).
Method B — Oval selection (fast, round plaques)
  1. Oval tool → drag over the plaque (Shift = circle); resize with handles.
  2. Ctrl+M → Area + ellipse Major/Minor.
Method C — Line across the diameter (one quick number)
  1. Straight‑line tool → draw edge‑to‑edge across the widest part.
  2. Ctrl+M → Length = that diameter (mm).
One chord only — for irregular plaques the Area method is fairer.
Area → tool‑style diameter in a spreadsheet: =2*SQRT(Area/PI()).

5 · Measure many plaques (ROI Manager)

1
Analyze → Tools → ROI Manager… (or press t after a selection).
2
Select a plaque → t to Add it. Repeat for each. Tick Show All + Labels to see numbered outlines.
3
Measure (all ROIs at once). For a movie/stack use More ▸ Multi Measure. More ▸ Save… writes RoiSet.zip so you can reload and re‑measure any time.
Redirect trick: draw/trace ROIs on the background‑subtracted image, but set Set Measurements → Redirect to: <raw original> so the gray values come from the untouched pixels — clean densitometry without altering your data.

6 · Automatic detection (Analyze Particles)

Threshold → Analyze Particles (area + intensity for every plaque)
  1. Image → Adjust → Threshold… (Ctrl+Shift+T) → slide so only the plaques highlight (don't Apply if you want to keep the grayscale for intensity — Analyze Particles can read a thresholded overlay while measuring the underlying gray).
  2. Analyze → Analyze Particles… → Size (mm²) range to drop specks, Circularity 0.4–1.0, tick Display results, Add to Manager, Exclude on edges, Summarize.
  3. OK → area + (with intensity ticked) mean gray for every plaque, plus a Summary row (count, total area, mean).
Top‑lit phone photos threshold poorly (uneven light, halos) — the manual wand/ROI route is usually more reliable. This gap is exactly why Plaque Toolkit's detector exists.

7 · Gray value & turbidity — the clarity measurement

A plaque's gray value is how bright/dark it is, which tells you how clear vs turbid it is. This is the Fiji equivalent of Plaque Toolkit's MEAN_GRAY and TURBIDITY_REL.

Do this on a single‑channel image (step 1). On an RGB image ImageJ measures a weighted brightness that depends on Edit → Options → Conversions — convert to 8‑bit or split channels so "gray value" is unambiguous.

Measure a plaque's brightness

1
Select the plaque (wand/oval) → Ctrl+M. Read Mean (mean gray, 0–255), Min/Max, StdDev, IntDen/RawIntDen.
  • Mean — average clarity of that plaque (≈ the tool's MEAN_GRAY).
  • RawIntDen — sum of all pixel values = total "clearing signal" (scales with size × depth).
  • StdDev — uniformity; a mottled/turbid plaque has higher StdDev than a cleanly lysed one.

Turn it into relative turbidity (like the tool)

2
Select a clean patch of the surrounding lawn (no plaques) → Ctrl+M → note its Mean = lawn brightness.
3
Compute relative turbidity = plaque Mean ÷ lawn Mean. ~1 → as opaque as the lawn (turbid); much lower (transmitted light) → clear. This mirrors TURBIDITY_REL.
Doing a whole plate? Add the lawn patch and every plaque to the ROI Manager, Measure once, then divide each plaque's Mean by the lawn Mean in your spreadsheet.

8 · See the clearing — profiles & histograms

Intensity profile across a plaque

1
Straight‑line tool → draw a line across a plaque (through the centre) → Ctrl+K (Analyze → Plot Profile).

You get a brightness‑vs‑position plot: a dip (top light) or peak (transmitted) at the plaque — its depth is the clearing strength, its width is the diameter. Click List/Save to export the profile, or Live to scrub across plaques.

Histogram of a plaque

Select a plaque → Ctrl+H (Analyze → Histogram) → the distribution of gray values inside it, with Mean / Mode / Min / Max / StdDev. List exports the bins.

9 · Toward optical density (advanced)

Gray value isn't optical density — but you can approximate. Analyze → Calibrate… → Function Uncalibrated OD maps each 8‑bit pixel to OD = log₁₀(255 / value), so Mean then reads as apparent OD. For calibrated OD you image a step‑tablet / known OD standards in the same shot and fit a curve (Rodbard) in that same dialog.

Same caveat as Plaque Toolkit. From top‑lit phone photos this is apparent OD (the phone tone‑maps the image) — treat it as relative / screening‑grade. True absorbance needs a back‑lit transilluminator (even, diffuse light), a locked exposure, and ideally RAW capture.

10 · Do it fast — record a macro & batch

1
Plugins → Macros → Record…, then do your steps once (open, set scale, measure). Fiji writes the macro code for you.
2
Save it, then run over a folder with Process → Batch → Macro… (pick the input folder, paste the recorded code, Process).

A minimal “measure everything already in the ROI Manager, on the current image” macro:

run("Set Measurements...", "area mean min standard integrated shape feret's redirect=None decimal=3");
roiManager("Measure");
saveAs("Results", getDirectory("home") + "fiji_results.csv");
You can also drag a folder of plates in with File → Import → Image Sequence… to make a stack, then ROI Manager ▸ Multi Measure across all slices at once.

11 · Save & analyse

  • Results table → File → Save As… → .csv (one per plate; Clear Results between plates).
  • ROI set → ROI Manager More ▸ Save… (RoiSet.zip) — your reusable, re‑measurable outlines.
  • Distributions → Results → Distribution… for a quick histogram of any column (e.g. diameter).
  • Scale‑bar figure → Analyze → Tools → Scale Bar… (works because the image is calibrated).

12 · Compare to Plaque Toolkit

Measure the same plate in Plaque Toolkit and Export all → data_<plate>.csv. In a spreadsheet:

  1. Size: convert Fiji Area → =2*SQRT(Area/PI()); match by number; difference vs the tool's DIAMETER_MM → mean bias + 95% limits (Bland–Altman).
  2. Turbidity: Fiji (plaque Mean ÷ lawn Mean) vs the tool's TURBIDITY_REL — check they trend together.
  3. Detection (optional): your Fiji ROI count vs the tool's count on the same plate.
Those two sentences — size agreement and turbidity agreement — are the validation for your Methods. See docs/METHODS_TEMPLATE.md and docs/PUBLISHING_CHECKLIST.md.

FAQ

My gray values look wrong / change when I recolour.

You're measuring an RGB image. Convert to 8‑bit (or split channels) first so "gray value" is a single, defined number. RGB "brightness" depends on Edit → Options → Conversions (weighted vs unweighted).

Should turbidity be a ratio or a difference?

Plaque Toolkit uses the ratio (plaque ÷ lawn), so use that to compare. A difference (lawn − plaque) is also fine if you're consistent and state it.

Uneven lighting is skewing my means.

Process → Subtract Background… on a duplicate, and/or always measure a local lawn patch next to each plaque rather than one global lawn value.

Which diameter should I report?

To match the tool, the area‑equivalent diameter from Area. Feret / ellipse‑Major are fine alternatives — state which and stay consistent.

Halo vs sharp clear zone?

Decide once. The tool's size is halo‑inclusive (convex hull), so trace the halo for the closest match; measure the sharp zone if that's your definition — and say which in Methods.
